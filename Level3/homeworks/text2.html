Frontend შესაქმნელად გამოყენებული პროგრამირების ენები

HTML (HyperText Markup Language)

შეიქმნა 1991 წელს ტიმ ბერნერს-ლის მიერ.

გამოიყენება ვებსაიტის სტრუქტურისთვის (ტექსტი, სურათი, ღილაკი და სხვ.). არის მარკირების ენა (არა პროგრამირების)

.

CSS (Cascading Style Sheets)

შეიქმნა 1996 წელს.

გამოიყენება HTML-ის გაფორმებისთვის (ფერები, ზომები, განლაგება). საშუალებას იძლევა ერთი სტილებით ბევრი გვერდი გააფორმო.

JavaScript

შეიქმნა 1995 წელს ბრენდან აიქის მიერ.

არის პროგრამირების ენა, რომელიც ამატებს დინამიურობას (ანიმაციები, ღილაკის დაჭერის რეაქცია, ფორმების შემოწმება). დღესაც ძალიან აქტუალურია, თითქმის ყველა საიტზე გამოიყენება.
